# Mode guides

Vanchor-NG is built around **modes** — each is a distinct autopilot behaviour you
pick from the on-screen **mode rail** (and the **More 🎣** flyout for the fishing
patterns). Only one mode is active at a time; **Stop** always drops back to Manual
with zero thrust. These guides explain, for a boater, what each mode does, when to
use it, and exactly which on-screen controls drive it.

## Station-keeping & basics
- **[Manual](manual.md)** — direct thrust + steering; you're the helm.
- **[Heading hold](heading-hold.md)** — hold a compass heading like a real autopilot.
- **[Anchor hold / Spot-Lock](anchor-hold.md)** — drop a virtual anchor and hold the
  spot (with drift anticipation, a nudge jog, and an optional learned "Smart" variant).
- **[Drift](drift.md)** — hold a heading while drifting at a gentle target speed.

## Navigation & routing
- **[Route / Waypoint navigation](route.md)** — build a route or tap "take me here";
  smart water-only routing (Fastest / Along-shoreline), Loop, Patrol, GPX, around-island.
- **[Work Area](work-area.md)** — visit a set of spots, hold station at each, and
  advance (timed or via a big "Go to next spot" button) — with an optional per-spot heading.
- **[Area survey ("map mode")](survey.md)** — lawnmower coverage over a drawn area.
- **[Follow APB](follow-apb.md)** — steer from an external route source (chartplotter / phone app).

## Fishing patterns (More 🎣)
- **[Contour follow](contour-follow.md)** — hold a chosen depth contour from the live sounder.
- **[Circle / Orbit](orbit.md)** — loop a marked point at a set radius (CW/CCW).
- **[Trolling pattern](trolling.md)** — a sinusoidal S-curve weave at a held speed.

---

> Safety floor (every mode): **Stop** cuts thrust immediately, and the safety
> governor (thrust slew limiting, shallow/no-go auto-stop, link-loss hold,
> loss-of-fix failsafe, anchor-drag alarm) runs underneath whatever mode is active.
